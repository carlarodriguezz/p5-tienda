# Práctica 5 - Consumo API REST

## Descripción
En esta práctica se ha conectado un frontend desarrollado en HTML, CSS y JavaScript con un backend en Java (Spring Boot) mediante una API REST.
Se utilizan llamadas asíncronas con `fetch` para consumir los endpoints y mostrar los datos dinámicamente en la web.

## Funcionalidades
- Crear carrito (POST)
- Añadir productos al carrito (POST)
- Mostrar productos del carrito (GET)
- Eliminar productos del carrito (DELETE)
- Manipulación del DOM con JavaScript

## Despliegue
El frontend está disponible en:
https://carlarodriguezz.github.io/p5-tienda/

## Backend
El backend se ejecuta en local en:
http://localhost:8080
Las funcionalidades del carrito requieren que el backend esté activo.
